# Overview

English Quest merupakan aplikasi web edukasi yang dikembangkan sebagai
Final Project. Aplikasi ini bertujuan untuk membantu pengguna dalam
mempelajari bahasa Inggris melalui pendekatan pembelajaran interaktif.

English Quest dirancang agar mudah digunakan oleh pelajar pemula hingga
menengah dengan tampilan sederhana dan fitur yang mudah dipahami.

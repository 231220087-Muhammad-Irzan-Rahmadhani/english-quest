# English Quest

English Quest adalah aplikasi web edukatif interaktif yang bertujuan
membantu pengguna belajar bahasa Inggris melalui cerita, kuis,
dan latihan kosakata.

## Tujuan
- Meningkatkan kosakata bahasa Inggris
- Melatih kemampuan membaca
- Menyediakan kuis interaktif

## Fitur
- Word of the Day
- Story Library
- Quiz
- Ranking
- Study Mode

## Teknologi
- HTML
- CSS
- JavaScript
